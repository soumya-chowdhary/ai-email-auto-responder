# ai-email-auto-responder
AI-powered system that reads emails and generates smart replies automatically using Gemini API
🚀 Features
📩 Automatically reads unread emails from inbox
🤖 Generates smart, professional replies using AI (Gemini API)
🔐 Allows replies only to trusted/allowed senders
⏰ Works within defined working hours
⚡ Fast and lightweight automation using Python
🧠 How It Works
1.Connects to your email using IMAP
2.Fetches the latest unread email
3.Checks if the sender is allowed
4.Extracts the email content
5.Uses AI to generate a polite reply
6.Sends the response automatically via SMTP
